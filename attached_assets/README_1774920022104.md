# TikTok診断ツール — Replit セットアップ手順

## ファイル一覧

| ファイル | 説明 |
|----------|------|
| setup.py | 全ファイル自動生成スクリプト |
| tiktok-shindan.html | 診断ツール（static/index.htmlに配置） |
| admin.html | 管理画面（static/admin.htmlに配置） |

---

## 手順

### 1. Replitにファイルをアップロード

以下の3ファイルをReplitのルートにアップロード:
- setup.py
- tiktok-shindan.html
- admin.html

### 2. setup.pyを実行

Replit の Shell で:
```
python setup.py
```

main.py / database.py / requirements.txt / .env.example が自動生成される。

### 3. HTMLファイルを配置

```
cp tiktok-shindan.html static/index.html
cp admin.html static/admin.html
```

### 4. .envファイルを作成

```
cp .env.example .env
```

.envを編集:
```
CLAUDE_API_KEY=sk-ant-xxxxxxxxxx
DATABASE_URL=postgresql://user:pass@ep-xxx.neon.tech/neondb
```

### 5. パッケージインストール

```
pip install -r requirements.txt
```

### 6. 起動

```
python main.py
```

---

## Neon DB 設定

1. https://neon.tech でプロジェクト作成
2. Connection string をコピー
3. .envのDATABASE_URLに貼り付け
4. テーブルは初回起動時に自動作成される

## VPS (Hostinger) へのデプロイ

```bash
# サーバーにアップロード後
pip install -r requirements.txt
pm2 start main.py --interpreter python3 --name tiktok-shindan
```
